# House Price Prediction in Bengaluru

## Overview
This project predicts house prices in Bengaluru using machine learning. It involves data preprocessing, feature engineering, outlier handling, and a comparative analysis of regression models to identify the best-performing model based on R-squared scores.

## Key Features
- Preprocessing and cleaning of real estate data.
- Feature engineering to extract meaningful insights.
- Handling outliers using statistical methods (IQR, Z-score).
- Comparative analysis of regression models (Linear Regression,Random Forest, Gradient Boosting).

## Dataset
The dataset contains information about house prices in Bengaluru, including features like location, area, number of bedrooms, and more.

## Requirements
- Python 3.x
- Pandas, NumPy, Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook
